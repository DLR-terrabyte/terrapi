
# terrapi

Terrabyte API CMD Tool 

## Usage

```
Usage: terrapi [OPTIONS] COMMAND [ARGS]...
```

## Arguments


## Options

* `debug`:
    * Type: BOOL
    * Default: `False`
    * Usage: `--debug`

    be more verbose



* `help`:
    * Type: BOOL
    * Default: `False`
    * Usage: `--help`

    Show this message and exit.
